# litmus
시약의 pH농도에 따른 리트머스 종이 색상 변화 시각화 모델
# 개발 환경
- 언어: c#
- 개발 Tool: Unity
# 팀원
4인 개
이름|엄민호|백진우|안성민|이윤환|
---|---|---|---|---|
역할|코드 총괄, ph scale과 시약 정보 연동|시약 농도에 따른 ph 값 알고리즘 구현|버튼 구현, 컬러모델 알고리즘 작성|인터페이스 구현, 시약 번짐 모델 구현|
